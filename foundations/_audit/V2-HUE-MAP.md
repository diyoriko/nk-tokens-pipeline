## Notes per hue
- **violet** — richest: 3 marketing families (Persian Indigo + Daisy Bush + Melrose) + Magnolia collapse into ONE ramp. Real coverage almost full; Main = #6D46FC. Collisions at light steps (3 near-white lights → pick one, others fold).
- **blue** — only LIGHT real colours (Patterns Blue, Periwinkle family). Main 🔒 = Periwinkle #C6D2FF (light). All mid/dark steps **GEN** (no dark brand blue exists). Bimodal note: Patterns Blue H234 vs Periwinkle H273 — wobble lives only in light tints.
- **magenta** — Heliotrope only: Light/Mid/Main/Dark real, rest GEN.
- **coral** — Persimmon: good light→mid coverage, darks GEN.
- **lemon** — Main #FFE60A is light (L91); tints above are near-white GEN, all dark steps GEN gold/olive (yellow is intrinsically light).
- **green** — Main 🔒 = Lime #79FD7F (per LDS) but Apple #31C838 (heavy code use) + Snake live as darker steps; near-full real coverage.
- **orange** — Orange family Light/Mid/Main/Dark real, deep darks GEN.
- **grey** — scale, no Main. Add GEN /900 #1A1A1A (true near-black, on-lemon anchor); Black #2C2A33 = /800.
